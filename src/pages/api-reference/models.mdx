# Models

SelamAPI provides access to four specialized AI model families, each fine-tuned for specific use cases and optimized for performance.

## Selam-Coder
- **Specialization**: Advanced code generation, debugging, and software architecture.
- **Capabilities**: Multi-language programming, code review, technical documentation, algorithm optimization.
- **Best for**: Software development, code analysis, debugging assistance, technical writing.
- **Context Window**: 128K tokens.

## Selam-Thinking
- **Specialization**: Deep analytical thinking, complex problem-solving, and step-by-step analysis.
- **Capabilities**: Logical reasoning, research analysis, strategic planning, critical thinking.
- **Best for**: Business analysis, research tasks, decision-making support, academic writing.
- **Context Window**: 128K tokens.

## Selam-Plus
- **Specialization**: Comprehensive assistance across a wide range of tasks.
- **Capabilities**: Creative writing, general conversation, content creation, summarization.
- **Best for**: Content creation, customer support, general assistance, creative projects.
- **Context Window**: 128K tokens.

## Selam-Turbo
- **Specialization**: Rapid responses with optimized performance.
- **Capabilities**: Quick answers, real-time assistance, high-throughput applications.
- **Best for**: Real-time applications, chatbots, instant responses, high-volume processing.
- **Context Window**: 32K tokens.
