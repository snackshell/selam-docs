# Overview

SelamAPI is a state-of-the-art AI platform providing access to fine-tuned language models through an OpenAI-compatible API. Built with Next.js, featuring Better Auth for authentication, and powered by a Neon PostgreSQL database.

Core features include an OpenAI-Compatible SDK, four specialized AI model families, enterprise authentication, advanced rate limiting, vision capabilities, streaming responses, and function calling.
